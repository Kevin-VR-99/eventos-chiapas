# Guía para colaborar

## Reglas para nombrar ramas

- rama-configuracion
- rama-inicio
- rama-eventos
- rama-contacto
- rama-estilos-js

## Reglas para mensajes de commit

Los mensajes deben ser claros y descriptivos.

Ejemplos:

- Agregar estructura inicial del sitio web
- Agregar formulario de contacto
- Agregar estilos del sitio
